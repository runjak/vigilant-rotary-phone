vigilant-rotary-phone
===

Toying around inspired by [1, 2].

Usage
---

* Install dependencies with `yarn install`
* Build `.js` into `dist` using `yarn build` or `yarn build:watch`
* Run tests using `yarn test` or `yarn test:watch`
